R4D is a free access on-line portal containing the latest information about research funded by DFID, including details of current and past research in over 30,000 project and document records. This web application widget uses open linked data from DFID, R4D and IATI to make that material more available for reuse on other websites.

To use, select a Project ID - the unique DFID Aries code, to generate a widget containing dynamically updated R4D content about the project and its research outputs. We encourage the use of the code on DFID-funded project sites and elsewhere.
